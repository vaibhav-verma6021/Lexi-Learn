# Welcome to our SEPM project

📘 Project Description

LexiLearn is an AI-powered learning platform designed to make studying more interactive, personalized, and engaging. Built using a modern full-stack setup and deployed via an AI-assisted development platform, the application leverages intelligent tools to help users better understand complex concepts and improve their learning experience.

The platform focuses on simplifying education by combining intuitive UI with AI-driven assistance. Users can explore topics, break down difficult concepts, and learn at their own pace through guided explanations and interactive content.

🚀 Features
AI-Assisted Learning
Get step-by-step explanations and deeper understanding of complex topics.
Interactive Experience
Learn through dynamic content instead of static reading.
Personalized Study Flow
Adapt learning based on user needs and pace.
Modern UI/UX
Clean, responsive interface for a smooth experience across devices.
🧠 Built With
Frontend: TypeScript, Tailwind CSS, react
Backend & Services: Supabase
AI Integration: AI-powered workflows via Lovable platform
Testing: Playwright
